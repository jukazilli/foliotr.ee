# Legado Documental

Status: historical context  
Last updated: 2026-04-27

## Conteudo

Esta pasta guarda documentos anteriores a aplicacao do Metodo Estrutural Integrado.

O lote inicial esta em:

`pre-integrado-2026-04-27/`

## Regra

Arquivos de legado podem ser usados para entender historico, decisoes passadas e referencias. Eles nao sao fonte primaria quando conflitarem com:

1. codigo real;
2. documentos ativos em `docs/00_*` a `docs/12_*`;
3. contratos e backlog atuais.

## Como usar

Antes de implementar algo baseado em documento legado:

1. verificar se existe contrato ativo equivalente;
2. comparar com o codigo real;
3. registrar lacuna se houver divergencia;
4. atualizar a documentacao ativa antes de codar.
