# Carousel

A themed card carousel with two visual modes — warm cinematic **Fantasy** and dark atmospheric **Cosmos**. Built entirely through conversation with [Claude Code](https://claude.ai/code).

## Themes

**Fantasy** — Cream background with film grain, dark cinematic cards with video backgrounds, warm shadows. Cards fan diagonally down-right.

**Cosmos** — Pure black with a static star field, dark gradient cards with cool glow borders. Cards fan diagonally up-left.

Toggle between themes with the pill switch in the top-right corner.

## Run locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
